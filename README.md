Это таймер времени для планирования задач и наглядного представления сколько свободного времени еще осталось, прежде чем пойти ко сну и/или завершить рабочий день.

Принцип работы:
1) Задаем время в которое завершаем деятельность (поскольку делал для себя, то начало в 13:01 и завершение в 23:58. Заверщение не позже 23:58 по двум причинам. Первая это не ложиться после двенадцати, вторая причина это не писать логику с временем после полуночи)
2) Добавляем задачу, проставляем прогнозируемое время выполнения
3) При выполении задачи нажимаем play на соответствующей задачи, так запускается отсчет времени выполнении задачи. (Дополнительно я включаю на телефоне таймер, чтобы визуально видеть сколько времени на задачу у меня осталось, пока ее, задачу, делаю)
4) Если в ходе дня таймер свободного времнеи становится сильно отрицательным, то какая-то  из задач удаляется из списка и возвращается обратно в список дел (не реализовано в рамках таймера. Список дел в отедбном блокноте)
