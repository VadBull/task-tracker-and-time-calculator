## Запуск

### В Docker (frontend + backend + postgres)

```bash
docker compose up --build
```

Фронтенд будет доступен на `http://localhost:5173`, backend — на `http://localhost:3001`.

### Локально (без Docker)

```bash
npm i
npm run sync
npm run dev -- --host
```

## Backend (Spring + Postgres)

The backend replaces `sync-server.js` and provides the same `/state` REST and WebSocket interface on port `3001`.

Это таймер времени для планирования задач и наглядного представления сколько свободного времени еще осталось, прежде чем пойти ко сну и/или завершить рабочий день.

Принцип работы:
1) Задаем время в которое завершаем деятельность (поскольку делал для себя, то начало в 13:01 и завершение в 23:58. Завершение не позже 23:58 по двум причинам. Первая это не ложиться после двенадцати, вторая причина это не писать логику с временем после полуночи)
2) Добавляем задачу, проставляем прогнозируемое время выполнения
3) При выполении задачи нажимаем play на соответствующей задачи, так запускается отсчет времени выполнении задачи. (Дополнительно я включаю на телефоне таймер, чтобы визуально видеть сколько времени на задачу у меня осталось, пока ее, задачу, делаю)
4) Если в ходе дня таймер свободного времени становится сильно отрицательным, то какая-то  из задач удаляется из списка и возвращается в список дел (не реализовано в рамках таймера. Список дел в отдельном блокноте)
